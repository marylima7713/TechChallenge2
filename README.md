# Descrição do projeto

O presente projeto tem como objetivo desenvolver um pipeline completo com análise e modelagem preditiva de um conjunto de dados relacionados às variáveis do vinho tinto português. O dataset utiliza o Wine Quality, contendo 1.018 amostras de vinhos com 11 características físico-químicas (como teor alcoólico, acidez volátil, pH, sulfatos, entre outras) e uma nota de qualidade atribuída por especialistas (de uma escala de 3 a 8).

# Problema
O problema foi formulado como uma tarefa de classificação binária supervisionada, onde visa desenvolver um modelo de classificação binária capaz de prever se um vinho é de alta qualidade (nota ≥ 7) ou de baixa/média qualidade (nota < 7). Essa simplificação permite focar no desafio mais relevante para a indústria: identificar quais vinhos atendem ao padrão de alta qualidade. Dessa forma, o modelo atua como uma ferramenta estratégica para auxiliar vinícolas na triagem inicial, na redução de custos de análise sensorial e tomada de decisões em tempo real durante a produção.

O dataset apresenta um desequilíbrio significativo entre as classes, com aproximadamente 86% dos vinhos classificados como baixa/média qualidade e apenas 13,5% como alta qualidade. Exigindo a aplicação rigorosa de estratégias específicas de balanceamento.
